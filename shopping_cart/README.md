# 🛒 Redux Toolkit Shopping Cart Mini Project

This is a **mini practice project** built using **React + Redux Toolkit** to understand core Redux concepts.

---

# 🚀 Project Overview

This project demonstrates how to manage global state using Redux Toolkit by building a simple **Shopping Cart system**.

Users can:

* Add items to cart
* Remove items from cart
* View total price

---

# 🧠 Concepts Used

This project is built using the following Redux Toolkit concepts:

* Store
* Slice
* Reducer
* Action
* useSelector
* useDispatch
