# 🌈 Mini Background Changer App (React Learning)

## 📌 What I Built Today

Today I created a **Mini Background Changer App** using React.
This app changes the background color when clicking different buttons.

---

## 🎯 Features

* Change background color on button click
* Dynamic inline styling
* Used `useState` hook
* Simple and interactive UI
* Event handling in React

---

## 🧠 Concepts Used

* `useState` for managing background color
* Event handling (`onClick`)
* Dynamic style change in React
* Component re-render on state update
